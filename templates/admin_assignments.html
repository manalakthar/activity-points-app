ℹ️ How Mentor Assignment Works
Each semester: Assign each student to their faculty mentor for that semester.
When student submits: The app automatically routes the submission to their assigned mentor for that semester.
Semester change: Simply create a new assignment for the new semester — old assignments are kept for records.
No assignment found: If no assignment exists, the submission goes to any available mentor in the same department as a fallback.
